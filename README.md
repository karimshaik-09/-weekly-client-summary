# Weekly Client Conversation Analysis using Generative AI

## Overview
This project analyzes an 8-day conversation between a health coach and a client using Generative AI. The objective is to convert unstructured conversations into a structured weekly client summary that helps coaches quickly understand the client's progress.

## Features
- Weekly Client Summary
- Nutrition Adherence Analysis
- Exercise and Step Tracking
- Sleep Analysis
- Water Intake Monitoring
- Symptoms and Stress Assessment
- Engagement Level
- Key Barriers
- Pending Actions
- Risk and Attention Flags
- Coach Recommendations
- Supporting Evidence Classification

## Technologies Used
- Generative AI
- Prompt Engineering
- Natural Language Processing (NLP)

## Project Output
The project generates a structured weekly report from daily client conversations, highlighting important health insights and recommended next actions for the coach.
